# MES Directory Notes

This folder stores dated MES / ES daily review articles.

## File Naming

- Preferred pattern: `YYYYMMDD_daily.md`
- Example: `20260225_daily.md`

## Article Format

- Use YAML front matter (`layout`, `title`, `description`, `keywords`)
- Publish in bilingual Chinese-English format
- Prefer embedded TradingView chart image syntax:
  - `![...](https://www.tradingview.com/x/...)`
- Keep a clickable TradingView link below the image

## Navigation Updates

When adding a new MES daily review, also update:

- `../PriceAction.md` (`MES Daily Reviews`)
- `../index.md` (`Latest Updates`)
